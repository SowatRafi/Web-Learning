# Basic Database Concepts

|**Terminology**|**Meaning**|
|---------------|-----------|
|`Database`| Organized collection of **data**.|
|`Table`|A table is a collection of _related_ **data** entries and it consists of **columns** and **rows**.|
|`Column`|Set of **data** _values_ in a **table**.|
|`Row`|A _record_ in a **table**.|
|`Schema`|**Schema** is a _collection_ of **database** _object_.|
|`Primary Key`|**Unique** _identifier_, such as a driver license number.|
|`Foreign Key`|Used to reference a **primary** **key** in another **table**.|

|**Relational Database**|**Database with related tables**|
|-----------------------|--------------------------------|
|**Relational Database Management System** - **RDMS**|_Software_ that manages _database_ e.g. **Oracle**, **Microsoft SQL Server**, **MySQL**|
