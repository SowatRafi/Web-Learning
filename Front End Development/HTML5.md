**Lecture 01:**
_Basics_

`<br>` for Break in the line.
`<hr>` for a horizontal line.
`&nbsp;` for a non breaking space.
`lorem` for a dummy text.

* For bold a text we can use `<b>` or `<strong>` tags.
* For a Italic text we can use `<i>` or `<em>` tags.
* For a underline in the text we can use `<u>` or `<ins>` tags.
* Use `<s>` or `<del>` represents a range of text that has been deleted from a document. 

* `<a href="https://www.discudemy.com" target="_blank">` for adding a link. If we don't use _https://www.example.com_ then the link will search in the computer files. So, please use full path for a website link. `target="_blank"` for opening the link with a new window.

* `<table>` `<tr>` `<th>` `<td>` for tables `rowspan` & `colspane` are also important words.

* `<form>` for a form.
* `<textarea>` for a text.
* `<required>` `<select> -> <option>`
* `<div>` for grouping the elements.
* For HTML5 Media uses two tags. `<video>` `<audio>` then use `<souce src="">` for the video/audio path from the computer. And also use `controls` in the `<video> <audio>` tag for playing it properly.
* `<iframes>` for online videos, maps & website connecting.
* `<figure>` `<figcaption>` for image and it details.
* `<details>` --> For a question -> `<summary>` --> `<div>` `<p>` for answer.

**Lecture 02**
_Web Structure_

* For Header use `<header>` then seperate them with `<div>`
* For Nav use `<nav>` then `<ul>` or `<ol>`
* For banner use `<section>` then the images.
* For adding contents `<section>` `<article>` `<main>` then `<h1/p>`
* For a sidebar use `<aside>`
* For a footer use `<footer>` `<address>` if you want to give address then use `<address>` if you want to add email & phone then  `<p><a href="mailto:contact@something.com">contact@something.com</a></p>` `<p><a href="tel:+88009924"> For call</a></p>`
* For Copyright use `<small>` & `&copy;`

**Responsive Web Design**

* We have to use `<meta name="viewport" content="width=device-width, initial-scale=1.0">` this tag for responsive design.
